# 🚖 Ola Ride Analytics Dashboard | Power BI

An interactive Power BI dashboard built using the Ola ride booking dataset to analyze booking trends, revenue, cancellations, vehicle performance, customer behavior, and ratings. This dashboard helps identify operational insights and key business metrics through interactive visualizations.

---

## 📌 Project Overview

This project provides an end-to-end business intelligence solution for analyzing Ola ride booking data. The dashboard enables users to monitor ride volume, booking status, revenue, cancellation patterns, vehicle-wise performance, payment methods, and customer/driver ratings through multiple report pages. :contentReference[oaicite:0]{index=0}
